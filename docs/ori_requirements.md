# SuperMart简介

一个面向专业从业者的UGC平台，让各领域的专家能够：

- 将工作流程转换为Skill工作流
- 将领域知识转换为Agent Bot
- 通过AI辅助提升工作效率

**核心价值主张**：让专业从业者分享实战经验，让行业知识规模化复用。

# Skill

## 定义

https://resources.anthropic.com/hubfs/The-Complete-Guide-to-Building-Skill-for-Claude.pdf

## Skill的属性

### 简介

在上传Skill后使用AI自动生成简介，简介内容包含以下几个内容：

- Skill的核心功能
- Skill的适用场景
- 一个真实案例的使用示例
- 一个合理性结论

### 付费（待规划）

Skill在上传时，可选择是否是付费，付费Skill具有如下限制：

- 未为该Skill付费的用户无法查看Skill内容、无法评论、下载

### 分类

Skill按照如下类型划分类型：

- 创作运营
- 代码开发
- 平面设计

后续根据垂直细分领域划分更多类型

### 安全性说明

在上传Skill时使用AI自动生成，暂时参考：https://github.com/openclaw/clawhub/blob/main/convex/lib/securityPrompt.ts

- 一个安全性结论
- 可能存在的安全性风险

### 评价

- 用户可对每个Skill评价
- 访客无法对Skill进行评价

### 版本

拥有版本控制功能：

- 每个Skill发布时都应有版本说明
- 能展示当前Skill的版本号以及版本说明
- 有展示历史版本的地方，可以查看每个版本的发布说明，可以下载任何一个历史版本

### 内容

- 能够展示Skill中的目录结构
- 能够预览Skill中的每个文件

### Skill的版权说明

- 暂时定位为MIT-0权限

# 平台用户

## 基本信息

- 姓名
- 邮箱
- 简介
- 收藏Skill列表
- 上传的Skill列表
- 付费类收益（待规划）

## 用户操作

- 上传Skill
- 对SKill收藏、打分、评论
- 对Skill下方的评论进行反馈
- 能够删除自己的账户，但与SKill关联的收藏、打分、评论数据不会清除

## 用户类型

### 访客

- 访客没有用户操作的权限
- 访客可以浏览Skill列表及内容
- 访客可以下载Skill

### 已注册用户

- 已注册用户在登陆后有用户操作权限
- 除了用户操作权限，已注册用户还有访客的所有权限

# 平台软件运营

## SKill内容运营

### 收藏

- 用户可以对喜欢的Skill进行收藏，可以在用户基本信息中查看
- Skill应该展示当前被多少用户收藏

### 下载量

- 已注册用户或访客每点击一次下载，会增加一次Skill的下载量

### 浏览量

- 已注册用户或访客没浏览一次Skill内容，会增加一次Skill的浏览量
