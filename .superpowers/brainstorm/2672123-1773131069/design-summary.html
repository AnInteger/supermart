SuperMart MVP 技术方案总结

基于 Brainstorming 确定的技术架构

📋 技术决策清单

决策项 选择 理由
后端架构 Next.js API Routes + Python 简化部署，Python 专注 AI
数据库 Supabase (PostgreSQL) 内置认证、存储、搜索
用户认证 邮箱注册 MVP 简洁，零成本
前端 UI shadcn/ui + Tailwind 现代设计，高度可定制
AI 功能 用户自带 API Key 有 AI 能力，无平台成本
部署方案 Vercel → 国内云 快速验证，后期迁移

📊 核心数据模型

👤 User
  • id, email, name
  • avatar_url, bio
  • created_at, updated_at
📦 Skill/Agent
  • id, title, description
  • type (skill/agent)
  • category, tags
  • content (JSON)
  • author_id, status
⭐ Review
  • id, skill_id, user_id
  • rating (1-5)
  • comment
  • created_at
