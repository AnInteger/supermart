问题2：数据库部署方式

A

Supabase (推荐)

PostgreSQL + 认证 + 存储，免费额度充足，与 Next.js 集成极佳

B

Firebase

NoSQL (Firestore)，Google 生态，实时数据同步，国内访问可能较慢

C

自托管 PostgreSQL

完全控制，需要自己管理服务器和备份
